Informationen ueber die Mapper Applikation

Diese Seite ist nur fuer private Tester gedacht.
Die Informationen koennen sich ohne Ankuendigung aendern.
Application parameter: 
java Mapper pfad_zu_maps map_dir sprache
z.B.: java Mapper /home/mh14/mapper/maps mg.mud.de de
z.B.: java Mapper /home/mh14/mapper/maps 3k.org en
Baumstruktur: 
pfad_zu_maps
    ->map_dir
          ->map 
	  enthaelt die Anzahl der Maps und zu jeder Map name und id

# ist ein Platzhalter fuer die Id der Map
pfad_zu_maps
    ->map_dir
          ->map#.coord 
	  enthaelt anzahl der Knoten, alle Knoten und deren Koordinaten
pfad_zu_maps
    ->map_dir
          ->map#.exits 
	  enthaelt anzahl der Ausgaenge, fuer jeden Ausgang Beginn- und Endknoten
	  (wenn der ausgang eine Map-Umschaltung beinhaltet ist der Endknoten als -knodenid-1 und eine ZielMap angegeben
	  wenn es einen gibt, der Name des special exit

pfad_zu_maps
    ->map_dir
          ->map#.info 
	  enthaelt knoten und fuer jedes vergebene info schluesselwort den wert

Bei Erstbenutzung eine Kopie von pfad_zu_maps/default zum mapnamen machen

Application Kommandos

  • save (mapid|-name)+: speichert die aktuelle map oder die angegebenen Maps

    TF support
